# 第一章：LLVM本体 (LLVM Core)

本章深入分析LLVM核心编译器基础设施，包括其项目结构、C++源代码惯用法、CMake构建系统以及TableGen技术。通过本章的学习，读者将了解LLVM本体的设计理念、代码组织方式以及如何有效地使用和修改LLVM核心组件。

## 本章内容

1. **项目结构与组织** - 了解LLVM的目录结构和构建系统概览
2. **C++源代码惯用法分析** - 深入分析LLVM的C++编码风格和常用设计模式
3. **CMake构建系统详解** - 详细介绍LLVM的CMake模块、变量和宏
4. **TableGen详解** - 理解TableGen的工作原理和使用方法
5. **常见问题 (FAQ)** - 解答关于LLVM本体的常见疑问
